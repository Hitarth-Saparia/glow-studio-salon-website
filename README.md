# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Hitarth-Saparia/glow-studio-salon-website.git
```

### Navigate to the Project

```bash
cd glow-studio-salon-website
```

### Run Locally

Simply open:

```bash
index.html
```

or use VS Code Live Server.
